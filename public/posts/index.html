  • I'm Sticking With UBlue Sep 25, 2023 - 2 min read

    I’ve been using Universal Blue heavily for a week now and it’s a keeper. UBlue is an immutable operating system based on Fedora where you install nearly all of your software as either flatpaks or as packages running in containers. Those containers can be based on any of a dozen different distributions. You can install Fedora packages to your base OS similar to a traditional distrubution but the philosopy behind UBlue encourages keeping your base install light and clean. Amongst other things, having a small, light and immutable base system greatly reduces your vector for malware.
